# 📊 Análise de Dados Energéticos (Brasil)

## 📌 Visão Geral
Este projeto realiza a análise de dados energéticos relacionados ao preço e volume de gás em diferentes bacias no Brasil.

## 🛠 Ferramentas Utilizadas
- Python  
- pandas  
- matplotlib  

## 🔎 Análises Realizadas
- Limpeza e preparação dos dados  
- Análise de volume por bacia  
- Análise de preço por bacia  
- Análise temporal (séries ao longo do tempo)  
- Correlação entre preço e volume  

## 📊 Principais Insights
- Identificação das bacias com maior volume de produção  
- Observação da variação de preços entre diferentes regiões  
- Análise de tendências ao longo do tempo  
- Exploração da relação entre preço e volume
